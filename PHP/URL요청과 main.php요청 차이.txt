/shop/main을 리턴하는 경우:

클라이언트(웹 브라우저)는 /shop/main 경로로 다시 요청을 보냄
서버는 해당 경로에 대한 처리를 수행하여 적절한 응답을 반환
이 과정에서 브라우저는 새로운 요청을 보내므로, 주소 표시줄에 /shop/main이 표시됨
페이지를 다시 로드하는 것이므로, 이전에 출력된 JavaScript 코드나 경고창은 사라짐


main.php 파일을 열어서 출력하는 경우:

main.php 파일의 내용이 서버에서 읽혀와서 클라이언트(웹 브라우저)에게 직접 전송
서버에서는 main.php 파일을 열어서 그 내용을 클라이언트에게 반환
이 과정에서 클라이언트는 새로운 요청을 보내지 않으며, 주소 표시줄에는 이전 요청 주소가 유지됨
JavaScript 코드나 경고창이 main.php 파일에 포함되어 있다면, 클라이언트가 해당 파일을 로드할 때 실행됨
